Dice

1. clone the repo from github into your computer.
2. Run the file named specRunner on your preffered browers.
3. View the code with any text editor of your choice.

About the repo
The repo contains the jasmine which is what I have used to write TDD, the source code can be found inside the srs folder and the test specs could be found in the spec folder.


DiceFactory class 
The class takes in the sides in the constractor and has a method called MakeDie.
> MakeDie is the functionthat generates random values to make up a die.


DiceTester class 
The class takes in the sides in the constractor and has a method called dieTest
> dieTest is a function that tests if the die is weighted or fair.
