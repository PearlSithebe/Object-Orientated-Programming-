About the repo
The repo contains the jasmine which is what I have used to write TDD, the source code can be found inside the srs folder and the test specs could be found in the spec folder.

Bank account class takes the three paramemters being the bank acoount number, balance and account type. The class has 4 methods namemly
> Balance which is a function that calculate the account balance taking the interest rates and charges into account.
> Deposits which is the function that allows any money into the account and adding it to the balance.
> Withdrawals which is the function that allows money to be taken out of the account and subtracting it to the balance.
> Transfers which is the function that allows money to be taken out of the accountto another account and subtracting it to the balance.
